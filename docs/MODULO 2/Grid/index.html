Grid

¿Qué es CSS Grid?​

CSS Grid es una poderosa herramienta de diseño que permite crear diseños flexibles y complejos en una cuadrícula bidimensional. Con CSS Grid, puedes definir filas y columnas y luego colocar los elementos de tu página en las celdas de esa cuadrícula. Te proporciona un mayor control sobre la colocación de los elementos en comparación con otros métodos de diseño en CSS, lo cual lo convierte en una opción muy útil.

Contenedor de la cuadrícula:​

display: grid;: Esta propiedad se aplica al contenedor de la cuadrícula y establece su tipo de visualización como una cuadrícula.
grid-template-columns: Define el tamaño y el número de columnas en la cuadrícula.
grid-template-rows: Define el tamaño y el número de filas en la cuadrícula.
grid-template-areas: Permite definir áreas dentro de la cuadrícula.

Elementos de la cuadrícula:​

grid-column: Establece en qué columnas debe ubicarse un elemento o cuántas columnas debe ocupar.
grid-row: Establece en qué filas debe ubicarse un elemento o cuántas filas debe ocupar.
grid-area: Asigna un elemento a un área específica dentro de la cuadrícula.

Alineación y espaciado:​

justify-items: Alinea los elementos horizontalmente dentro de sus celdas.
align-items: Alinea los elementos verticalmente dentro de sus celdas.
justify-content: Alinea la cuadrícula horizontalmente dentro de su contenedor.
align-content: Alinea la cuadrícula verticalmente dentro de su contenedor.
gap: Establece el espacio entre las celdas de la cuadrícula.

Estas son solo algunas de las propiedades más comunes utilizadas en CSS Grid. Sin embargo, CSS Grid ofrece muchas más propiedades y funciones avanzadas, como la creación de áreas implícitas, el cambio del orden de visualización de los elementos utilizando order, el posicionamiento de elementos utilizando grid-column-start, grid-column-end, grid-row-start y grid-row-end, entre otros.

Aquí tienes un ejemplo más completo para comprender mejor cómo funciona CSS Grid:y columnas:

<!DOCTYPE html>
<html>
<head>
  <style>
    .grid-container {
      display: grid;
      grid-template-columns: repeat(3, 1fr); /* Define 3 columnas del mismo tamaño */
      grid-template-rows: 100px 200px auto; /* Define la altura de las filas */
      gap: 10px; /* Espacio entre elementos */
    }

    .grid-item {
      background-color: #ddd;
      padding: 20px;
    }
  </style>
</head>
<body>
  <div class="grid-container">
    <div class="grid-item">Elemento 1</div>
    <div class="grid-item">Elemento 2</div>
    <div class="grid-item">Elemento 3</div>
    <div class="grid-item">Elemento 4</div>
    <div class="grid-item">Elemento 5</div>
    <div class="grid-item">Elemento 6</div>
    <div class="grid-item">Elemento 7</div>
  </div>
</body>
</html>



En este ejemplo, hemos creado una cuadrícula con tres columnas utilizando grid-template-columns. La función repeat(3, 1fr) indica que queremos repetir 1fr (una fracción del espacio disponible) tres veces para definir las columnas.

Luego, hemos definido la altura de las filas usando grid-template-rows. Hemos especificado tres valores: 100px, 200px y auto. La primera fila tendrá una altura fija de 100px, la segunda fila tendrá una altura fija de 200px y la tercera fila se ajustará automáticamente según el contenido.

Hemos agregado más elementos a la cuadrícula para mostrar cómo se ajustan. Observa cómo se colocan automáticamente en la cuadrícula siguiendo el orden de los elementos en el HTML.

Los elementos .grid-item tienen un color de fondo y un relleno para resaltarse visualmente.

Puedes experimentar con el número de columnas, las alturas de las filas y los estilos para ver cómo afectan al diseño de la cuadrícula. Además, puedes utilizar propiedades adicionales como grid-column y grid-row para especificar ubicaciones específicas de los elementos en la cuadrícula.
